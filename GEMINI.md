## 4. КОНТЕКСТ ПРОЕКТА (Project Context)

### Обзор (Overview)
**Readen** — это легковесный, self-hosted EPUB ридер.
*   **Цель:** Удобное чтение книг и копирование глав для скармливания LLM.
*   **Стек:** Python 3.10+, FastAPI, SQLite, React (планируется), Tailwind CSS.

### Структура (Structure)
*   `backend/`: Современный бэкенд на FastAPI.
*   `legacy/`: Старая версия (монолитный скрипт + сервер).
*   `_data/`: Папки с обработанными книгами (pickle).

### Запуск (Running)
Проект использует `uv` для управления зависимостями.
```bash
uv run backend/app/main.py
```
